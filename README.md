# Simulateur de course de trot attelé

Une course de trot attelé rassemble 12 à 20 chevaux. Elle peut faire l’objet d’un tiercé, d’un quarté, ou d’un quinté. La course se déroule sur un hippodrome, d’une longueur de 2400m. Il est à noter que chaque cheval doit respecter l’allure du trot de bout en bout, le passage au galop entrainant sa disqualification.
